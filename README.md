![yt-dlp](https://img.shields.io/badge/powered%20by-yt--dlp-red)

![License](https://img.shields.io/badge/license-MIT-lightgrey)

A **portable PowerShell launcher** for **yt-dlp** that automatically installs and updates **yt-dlp** and **FFmpeg**, with a simple menu-driven interface.

No global installs. No PATH changes. Windows-only.

---

## Features

* One-click `.bat` launcher
* Automatic yt-dlp updates
* Automatic FFmpeg installation
* PowerShell 5 compatible
* Clean text-based menu
* Audio and video download modes
* Shows downloaded file paths

---

## Requirements

* Windows
* PowerShell 5+
* Internet connection

No admin rights required.

---

## Usage

1. Clone or download the repo
2. Double-click `launch.bat`
3. Choose **Download** or **Update All** from the menu

Downloads are saved to `Downloads/`.
